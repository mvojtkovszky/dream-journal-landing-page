# Dream Catcher — Landing Page

Static landing page for [Dream Catcher](https://dreamjournal.app).

## Structure

```
dream-journal-landing-page/
├── index.html   # Single-page site (all CSS inline)
└── images/      # App screenshots, badges, and SVG assets
```

## Stack

- Plain HTML/CSS — no build step, no dependencies
- Google Fonts (Roboto) via WebFont Loader
- App Store and Google Play download badges

## Usage

Open `index.html` directly in a browser, or serve with any static file server:

```sh
npx serve .
```
